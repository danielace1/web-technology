ex 1.4
Title:
Webpage with an image map of a computer system components

Objective:
To create a web page using HTML that displays computer system components as images. Each image should act as a clickable region using image mapping and navigate to the corresponding component information section within the same webpage.

Tools Required:
HTML5
Web Browser 
Text Editor
Image files of computer components 

Procedure:

* Open a text editor and create a new file with .html extension.

* Write the basic HTML structure using <html>, <head>, and <body> tags.

* Add a heading for the webpage title.

* Create a table to place images side by side.

* Insert images of CPU, motherboard, RAM, and hard disk using the <img> tag.

* Use the usemap attribute to connect images with their respective image maps.

* Define image maps using the <map> tag.

* Create clickable areas inside each map using <area> tags.

* Link each area to its respective information section using internal links (#id).

* Create separate sections for CPU, motherboard, RAM, and hard disk using <h3> tags.

* Save the file and open it in a web browser.

* Click each image and verify it navigates to the correct information section.

Code: 

<!doctype html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <meta name="viewport" content="width=device-width, initial-scale=1.0" />
    <title>Computer System Components - Image Map</title>
  </head>

  <body>
    <body>
      <h2 align="center">Computer System Components - Image Map</h2>

      <center>
        <table>
          <tr>
            <td>
              <img src="./assets/img/cpu.png" usemap="#cpu" width="200" />
              <p align="center">CPU</p>
            </td>

            <td>
              <img
                src="./assets/img/motherboard.png"
                usemap="#mb"
                width="200"
              />
              <p align="center">Motherboard</p>
            </td>

            <td>
              <img src="./assets/img/ram.png" usemap="#ram" width="200" />
              <p align="center">RAM</p>
            </td>

            <td>
              <img src="./assets/img/hdd.png" usemap="#hdd" width="200" />
              <p align="center">Hard Disk</p>
            </td>
          </tr>
        </table>
      </center>

      <map name="cpu">
        <area shape="rect" coords="0,4,512,326" href="#cpu" />
      </map>

      <map name="mb">
        <area shape="rect" coords="0,0,6117,6009" href="#mb" />
      </map>

      <map name="ram">
        <area shape="rect" coords="9,13,505,504" href="#ram" />
      </map>

      <map name="hdd">
        <area shape="rect" coords="22,27,6117,6009" href="#hdd" />
      </map>

      <hr />

      <h3 id="cpu">CPU</h3>
      <p>CPU (Central Processing Unit) is the brain of the computer.</p>

      <h3 id="mb">Motherboard</h3>
      <p>Motherboard connects all hardware components together.</p>

      <h3 id="ram">RAM</h3>
      <p>RAM stores temporary data and running programs.</p>

      <h3 id="hdd">Hard Disk</h3>
      <p>Hard Disk stores data permanently.</p>
    </body>
  </body>
</html>


Learning Outcome:
* Create image maps using HTML
* Place images side by side using tables
* Use internal page navigation
* Understand <map> and <area> tags
* Develop simple interactive webpages
* Improve basic HTML skills
