Ex 2.3


Title:

Design and Development of a Responsive Restaurant Website using Advanced CSS Techniques


Objective:

To build a restaurant website using various CSS techniques such as parallax scrolling, CSS Grid layout, hover effects, transition effects, responsive image gallery, custom form controls, and CSS animations for page loading.


Tools Required:

HTML5

CSS3

Web Browser (Chrome, Firefox, etc.)

Text Editor (Notepad, VS Code, etc.)

Internet connection (for images)


Procedure:

Open a text editor and create a new file with .html extension.

Write the basic HTML structure using <html>, <head>, and <body> tags.

Add CSS inside the <style> tag to apply global styling and reset margins.

Create a navigation bar using <nav> and apply transition effects to links.

Design a full-screen hero section and apply a parallax scrolling effect using background-attachment: fixed.

Implement a fade-in animation for page loading using @keyframes.

Create a food menu section using CSS Grid and display images.

Apply hover effects to menu images using transform: scale().

Create a responsive image gallery using CSS Grid that adapts to different screen sizes.

Design a reservation form using custom styled <input> and <button> elements.

Apply media queries to make the webpage responsive for small screens.

Save the file and open it in a web browser.

Verify that animations, hover effects, and responsive layouts work correctly.



Code:


<!DOCTYPE html>
<html>
<head>
    <title>Restaurant Website</title>
    <style>
        * {
            margin: 0;
            padding: 0;
            box-sizing: border-box;
            font-family: Arial, sans-serif;
        }

        /* Page load animation */
        body {
            animation: fadeIn 1.2s ease-in;
        }

        @keyframes fadeIn {
            from { opacity: 0; }
            to { opacity: 1; }
        }

        /* Navigation */
        nav {
            background: black;
            padding: 15px;
            text-align: center;
        }

        nav a {
            color: white;
            margin: 0 20px;
            text-decoration: none;
            transition: 0.3s;
        }

        nav a:hover {
            color: orange;
        }

        /* Hero Section with Parallax */
        .hero {
            height: 100vh;
            background: url("https://images.unsplash.com/photo-1504674900247-0877df9cc836") center/cover no-repeat;
            background-attachment: fixed;
            display: flex;
            justify-content: center;
            align-items: center;
            color: white;
            font-size: 40px;
        }

        /* Menu Section */
        .menu {
            padding: 40px;
            text-align: center;
        }

        .menu-grid {
            display: grid;
            grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
            gap: 20px;
        }

        .menu-item img {
            width: 100%;
            transition: 0.3s;
        }

        .menu-item img:hover {
            transform: scale(1.1);
        }

        /* Gallery */
        .gallery {
            padding: 40px;
            text-align: center;
        }

        .gallery-grid {
            display: grid;
            grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));
            gap: 15px;
        }

        .gallery-grid img {
            width: 100%;
        }

        /* Reservation Form */
        .reservation {
            padding: 40px;
            text-align: center;
        }

        .reservation input, .reservation button {
            width: 250px;
            padding: 10px;
            margin: 10px;
            border-radius: 5px;
            border: 1px solid #999;
        }

        .reservation button {
            background: orange;
            color: white;
            border: none;
        }

        /* Responsive */
        @media (max-width: 600px) {
            .hero {
                font-size: 28px;
            }
        }
    </style>
</head>

<body>

<!-- Navigation -->
<nav>
    <a href="#">Home</a>
    <a href="#menu">Menu</a>
    <a href="#gallery">Gallery</a>
    <a href="#reserve">Reserve</a>
</nav>

<!-- Hero -->
<div class="hero">
    Welcome to Our Restaurant
</div>

<!-- Menu -->
<div class="menu" id="menu">
    <h2>Food Menu</h2><br>
    <div class="menu-grid">
        <div class="menu-item"><img src="https://images.unsplash.com/photo-1600891964599-f61ba0e24092"></div>
        <div class="menu-item"><img src="https://images.unsplash.com/photo-1550547660-d9450f859349"></div>
        <div class="menu-item"><img src="https://images.unsplash.com/photo-1565299624946-b28f40a0ae38"></div>
    </div>
</div>

<!-- Gallery -->
<div class="gallery" id="gallery">
    <h2>Food Gallery</h2><br>
    <div class="gallery-grid">
        <img src="https://images.unsplash.com/photo-1504674900247-0877df9cc836">
        <img src="https://images.unsplash.com/photo-1551218808-94e220e084d2">

        <img src="https://images.unsplash.com/photo-1567620905732-2d1ec7ab7445">

</div>

<!-- Reservation -->
<div class="reservation" id="reserve">
    <h2>Table Reservation</h2><br>
    <form>
        <input type="text" placeholder="Name"><br>

</body>
</html>


Learning Outcome:

Learn to create parallax scrolling effects using CSS.

Understand CSS Grid for responsive layouts.

Apply hover and transition effects using CSS.

Design responsive image galleries.

Create custom styled form controls.

Implement CSS animations using @keyframes.

Build a modern, responsive webpage layout.
